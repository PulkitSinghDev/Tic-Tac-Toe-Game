# Tic-Tac-Toe-Game
A Tic Tac Toe Command Line Game in Python by @PulkitSinghDev
## Setup & Installtion

Make sure you have the latest version of Python installed.

```bash
git clone <repo-url>
```

```bash
pip install -r requirements.txt
```

## Running The App

```bash
python main.py
```
